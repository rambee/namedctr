# namedctr
Named counter app, a Django based web app.

# Screenshot


# Installing the prerequisites (Development)

## Python > 3

$ brew install python3

## pip

$ brew postinstall python3

## Django

$ pip3 install django

# Execution

$ cd namedctr

$ python manage.py runserver

Type http://127.0.0.1/8000/ in the web browser
